Tratamiento saludable y natural de dolores

PARA CUELLO | ESPALDA | CINTURA | MUÑECA | MANO

DESCRIPCIÓN · USOS

Contiene hierbas secas y granos naturales que en combinación con el calor o frío emiten un efecto aromático relajante.

Con calor disminuye síntomas de artritis, dolores musculares, cólicos, gripes, favorece al buen funcionamiento de riñón y vejiga.

En frío alivia ojos inflamados, moretones, síndrome del Túnel Carpiano, fiebre, pies cansados, dolor de muelas, picaduras de insectos, estrés, dolor de cabeza y migraña.

Se puede calentar en microondas [períodos desde un minuto hasta sentir la temperatura deseada].

También se puede enfriar en el congelador / freezer [dentro de una bolsa para conservarla seca].

Almohadilla para cuello, cintura

Se recomienda para contracturas, dolores musculares, etc.

Almohadilla para muñeca

Para dolores y molestias por uso del mouse, túnel carpiano, etc.

Almohadilla para mouse

Para descansar mientras se usa el mouse.
